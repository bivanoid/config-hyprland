# Hyprland Config (Ax-Shell Edition)

Repositori ini berisi konfigurasi **Hyprland** yang saya gunakan, yang terintegrasi penuh dengan **Ax-Shell** dari [Axenide](https://github.com/axenide).

## Preview
![Preview](https://github.com/user-attachments/assets/6fe6fec2-f71f-455e-88ab-bdfd047e3260)

## Tech Stack
Konfigurasi ini dirancang dengan komponen berikut:
- **Window Manager:** [Hyprland](https://hyprland.org/)
- **Terminal:** [Foot Terminal](https://codeberg.org/dnkl/foot)
- **Shell & Components:** [Ax-Shell](https://github.com/axenide) (Mencakup bar, app launcher, notifikasi, dan wallpaper)
- **Teks Editor:**  Zed
- **Browser:** Brave Origin
